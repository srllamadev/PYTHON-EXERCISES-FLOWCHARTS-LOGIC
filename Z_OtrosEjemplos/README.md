# PresionarTecla_SinTeclado.py

Este script simula la pulsación de la tecla 'L' en el teclado 10,000 veces utilizando la biblioteca pynput.

## Requisitos

- Python 3.x
- Instalar las bibliotecas necesarias:
  ```
  pip install pynput
  ```

## Cómo ejecutar

Ejecuta el script con Python:
```
python PresionarTecla_SinTeclado.py
```

El script esperará 2 segundos antes de comenzar a simular las pulsaciones, con una pausa de 0.03 segundos entre cada una.

## Advertencia

Este script simula entradas de teclado. Úsalo con precaución para evitar interferencias no deseadas en otros programas.
