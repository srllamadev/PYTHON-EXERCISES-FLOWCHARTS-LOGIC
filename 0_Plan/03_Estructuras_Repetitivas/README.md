# 03 - Estructuras Repetitivas

## Descripción
Las estructuras repetitivas (bucles o loops) permiten ejecutar un bloque de código múltiples veces, lo que es fundamental para automatizar tareas repetitivas.

## Contenido

### 01_While_Loops
- Bucles con condición
- Control de flujo con while
- Bucles infinitos y cómo evitarlos

### 02_For_Loops
- Iteración sobre secuencias
- Bucles for con diferentes estructuras de datos
- Enumerate y zip

### 03_Break_Continue_Pass
- break: salir de un bucle
- continue: saltar a la siguiente iteración
- pass: placeholder sin operación

### 04_Range_Function
- Generación de secuencias numéricas
- range() con diferentes parámetros
- Uso en bucles for

## Objetivos de Aprendizaje
- Comprender cuándo usar while vs for
- Dominar el control de flujo en bucles
- Optimizar código con estructuras repetitivas
- Evitar bucles infinitos
