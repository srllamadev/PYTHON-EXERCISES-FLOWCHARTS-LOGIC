# 05 - Funciones

## Descripción General

Las funciones son bloques de código reutilizables que realizan una tarea específica. Son uno de los conceptos más importantes de la programación, permitiendo organizar, reutilizar y mantener el código de manera eficiente.

## Estructura del Módulo

### 📁 [01_Definicion_y_Llamada](01_Definicion_y_Llamada/)
Aprende a definir funciones propias y cómo invocarlas en tus programas.

**Temas clave:**
- Sintaxis `def`
- Cuerpo de una función
- Llamada a funciones
- Funciones sin parámetros

### 📁 [02_Parametros_y_Argumentos](02_Parametros_y_Argumentos/)
Domina la diferencia entre parámetros y argumentos, y los distintos tipos que existen.

**Temas clave:**
- Parámetros posicionales
- Argumentos por nombre (keyword arguments)
- Valores por defecto
- Parámetros obligatorios vs opcionales

### 📁 [03_Return_Values](03_Return_Values/)
Aprende a retornar valores desde una función y cómo usarlos.

**Temas clave:**
- Sentencia `return`
- Retorno de múltiples valores (tuplas)
- Funciones sin `return` (retornan `None`)
- Uso del valor retornado

### 📁 [04_Scope](04_Scope/)
Comprende el alcance de las variables dentro y fuera de las funciones.

**Temas clave:**
- Scope local vs global
- Palabra clave `global`
- Palabra clave `nonlocal`
- LEGB rule (Local, Enclosing, Global, Built-in)

### 📁 [05_Lambda_Functions](05_Lambda_Functions/)
Conoce las funciones anónimas de una sola expresión.

**Temas clave:**
- Sintaxis `lambda`
- Uso con `map()`, `filter()`, `sorted()`
- Cuándo usar lambdas vs funciones normales
- Limitaciones de las lambdas

### 📁 [06_Args_Kwargs](06_Args_Kwargs/)
Aprende a crear funciones que aceptan un número variable de argumentos.

**Temas clave:**
- `*args` - argumentos posicionales variables
- `**kwargs` - argumentos de palabra clave variables
- Combinación de parámetros
- Desempaquetado con `*` y `**`

### 📁 [07_Decoradores_Basicos](07_Decoradores_Basicos/)
Introduce el patrón decorador para extender el comportamiento de funciones.

**Temas clave:**
- Funciones como objetos de primera clase
- Funciones anidadas y closures
- Sintaxis `@decorador`
- Decoradores con `functools.wraps`

## Objetivos de Aprendizaje

Al completar este módulo, serás capaz de:

✅ Definir y llamar funciones propias  
✅ Usar parámetros, argumentos y valores por defecto  
✅ Retornar uno o múltiples valores  
✅ Entender el scope de variables (LEGB)  
✅ Escribir funciones lambda para casos simples  
✅ Crear funciones con `*args` y `**kwargs`  
✅ Aplicar decoradores básicos a tus funciones  

## Requisitos Previos

- Módulos 01 al 04 completados
- Manejo de listas, tuplas y diccionarios
- Estructuras condicionales y loops
